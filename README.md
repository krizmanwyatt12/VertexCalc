# VertexCalc

**VERTEX — Cost of the Status Quo & Value Model**

A live, interactive ROI model prepared for The Vertex Companies for the July 16, 2026 review.

## What it is

A single self-contained HTML page. Replace the default inputs with your own figures and every total — cost of standing still, year-one recovery, ROI, payback — updates live. The conservative floor reconciles to the July 2026 proposal ($290K to $530K year-one value).

## How to view

**GitHub Pages (recommended):** Settings > Pages > Source: Deploy from a branch, select main / root. The site publishes at https://krizmanwyatt12.github.io/VertexCalc/

**Raw (instant):** https://raw.githubusercontent.com/krizmanwyatt12/VertexCalc/main/index.html

## Brand
